下面是我使用过的计算机语言列表:

java
最初学习的主方向.基础比较扎实,后来学习了SSH等框架.现在这些早就微不足道.目前java的优势在于稳定性和安全性.企业更多的是用它来解决多并发,一致性,分布式计算等大型问题
c#
做客户端是非常快的,而且要比java的界面开发周期短,缺点是只能在windows系统上使用.学习起来比较简单,好多年没用了
javascript
早先用的是es3,目前已经到e6,7语法有了很大变化;另外对应的html,css不能算是开发语言,对应的模版库提供的编程也不能严格算语言
sql/plsql
但凡是个程序员都要会点数据库知识.sql现在还是主流,无非就是对数据CURD.难点的有触发器,索引,存储过程等.不过现在更流行的是nosql数据库,比如mongodb,hive等.可以存储更复杂类型的数据,可以说是以后大数据架构的核心类型的数据库.
actionscript
用了几年的flex开发,其本质就是基于flash 关键帧的脚本语言actionscript.这个语言本身也是挺好的.可惜flex框架已经没落,小公司都喜欢用第三方,这个框架已经没人维护,这个语言迟早也没人用的
python
用过linux/unix 系统的人都知道系统默认安装了python,因为很多工具包都要依赖它来管理.它本身也可以用来做服务器开发.spark框架更是用它来做大数据.好的语言基本都是跨平台的,它也不例外.
ruby
ruby 在我映像中就是用来做配置的.有点像shell脚本.不过要比shell灵活一些.做iOS开发用到的cocoapods工具,里面的Podfile 就是用ruby编写.用来直到编译器按照一定的依赖和条件进行项目编译
scala
基于JDK的扩展语言,语法大部分跟java是类似的,主要用来做大数据,特点是擅于进行并行计算
swift
苹果公司2014年初推出的一款开发语言,随着开源有很多人对它进行评估和改造.这个语言的特点是让代码简洁的不能再简洁.这其实没有什么.最期待的是它的彻底化的泛型和对并行计算的支持
